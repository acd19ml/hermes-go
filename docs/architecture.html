Registry 派发流程

1
DispatchTool(ctx, tc) → globalRegistry.Dispatch(ctx, tc)
2
Registry 按 tc.Name 查 map[string]ToolEntry
3
找到 → 调 entry.Handler(ctx, tc)  |  找不到 → {"error":"...","code":"unknown_tool"}
工具错误统一格式 {"error":"...","code":"..."}；code 枚举：unknown_tool · bad_arguments · path_escape · execution_error

echo  toolset: core

1
json.Unmarshal → args.Text
2
返回 ToolResult{Content: text}
解析失败 → IsError:true, code:"bad_arguments"

read_file  toolset: file

1
json.Unmarshal → args.Path
2
filepath.Abs(join(cwd, path)) · filepath.Rel 安全检查
3
os.ReadFile(abs) → 文件内容
路径逃出 cwd → code:"path_escape" · 读取失败 → code:"execution_error"

Toolset 分组 & 动态 Schema

core → echo
file → read_file
debugging → core + file（别名展开）
默认：enabledToolsets = ["core","file"]
Respond() 每次请求调 globalRegistry.GetSchemas(enabledToolsets) 动态构建 tools 数组，不缓存。新增工具只需在 init() 注册，所有已有 client 自动可见。
